# capcut-srt-desktop
capcut generate STR subtitles desktop tool

capcut-srt-desktop 是一款CapCut 生成SRT字幕工具，功能支持自定义生成输出位置以及字幕名称，方便大家快速的生成字幕节约时间！提高大家的工作效率！

目标版本只支持在Windows系统上进行使用，后期有时间会适配MacOs系统
